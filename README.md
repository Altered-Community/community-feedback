# Altered Community — Requests & Feedback

This repository tracks bugs, feature requests, and questions for the core services of the Altered community.

> **No GitHub account?** Submit your request directly from the website:
> **[alteredcore.org/pages/feedback](https://alteredcore.org/pages/feedback)**

---

## Services covered

- **Cards API** — card data, sets, factions
- **Decks API** — deck management and sharing
- **Auth (Keycloak)** — login and accounts
- **alteredcore.org** — the community website

## How to submit

Open an [issue](../../issues) and pick the template that fits:
- **Bug report** — something is broken
- **Feature request** — an idea or improvement
- **Question** — need clarification

Issues can be written in **French or English**.

---

Please be respectful and constructive.
